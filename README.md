# Egypt Adventures

## Overview
Egypt Adventures is an immersive game where players explore ancient pyramids, battle enemies, and uncover hidden treasures. This project is designed to provide an engaging gaming experience with dynamic mechanics and rich storytelling.

## Features
- **Dynamic Combat System**: Engage in battles with unique slot-based mechanics.
- **Exploration**: Navigate through deserts, pyramids, and other mysterious locations.
- **Customization**: Equip your character with various items and abilities.
- **Localization**: Supports multiple languages for a global audience.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Lila-dejavu/egypt_adventures.git
   ```
2. Navigate to the project directory:
   ```bash
   cd egypt_adventures
   ```
3. Open `index.html` in your browser to start the game.

## File Structure
- `index.html`: Entry point of the game.
- `style.css`: Contains all the styles for the game.
- `main.js`: Initializes the game and handles core logic.
- `js/`: Contains modular JavaScript files for different functionalities.
  - `core/`: Core game logic and utilities.
  - `combat/`: Combat-related mechanics.
  - `ui/`: User interface management.
  - `mixins/`: Reusable game logic components.
