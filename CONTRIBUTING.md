# Contributing to Egypt Adventures

Thank you for your interest in contributing to Egypt Adventures! We welcome contributions from the community to help make this game even better.

## How to Contribute

### Reporting Bugs
If you encounter a bug, please open an issue on GitHub with:
- A clear and descriptive title
- Steps to reproduce the issue
- Expected behavior vs. actual behavior
- Screenshots or error messages (if applicable)
- Your browser and operating system information

### Suggesting Enhancements
We welcome feature requests and enhancement suggestions! When proposing a new feature:
- Explain the problem the feature would solve
- Describe the proposed solution
- Consider alternative solutions
- Explain why this enhancement would be useful to most users

### Pull Requests
1. **Fork the repository** and create your branch from the main branch
   ```bash
   git checkout -b feature-name
   ```

2. **Make your changes**
   - Write clear, commented code
   - Follow the existing code style
   - Test your changes thoroughly

3. **Commit your changes**
   ```bash
   git commit -m "Add new feature"
   ```
   - Use clear and descriptive commit messages
   - Reference any related issues

4. **Push to your fork**
   ```bash
   git push origin feature-name
   ```

5. **Open a Pull Request**
   - Provide a clear description of the changes
   - Link any related issues
   - Explain how to test the changes

### Development Guidelines
- **Code Style**: Follow the existing code style in the project
- **Comments**: Add comments for complex logic
- **Testing**: Test your changes in multiple browsers
- **Localization**: If adding new text, ensure it's added to the localization system

### Code of Conduct
Please note that this project follows a Code of Conduct. By participating, you agree to uphold this code. Please report unacceptable behavior to the project maintainer.

## Getting Help
If you need help or have questions:
- Check existing issues and pull requests
- Open a new issue with your question
- Contact the maintainer at [Lila-dejavu](https://github.com/Lila-dejavu)

Thank you for contributing to Egypt Adventures!
